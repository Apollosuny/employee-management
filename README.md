### Git command
1. Add files changed to stage
```bash
git add .
```

2. Make commit message
```bash
git commit -m "your_msg"

3. Push code to github
```bash
// First time Push
git push -u origin your_branch
```

// Other
git push

4. Pull new code
```bash
git pull
```

5. Merge two branch
```bash
// merge branch_name to your current_branch
git merge branch_name
```
